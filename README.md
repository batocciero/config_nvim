# My Config backup

Dependencies:

- node
- ruby gem (solargraph, rufo)

#### Basic settings

- set hidden
- set number
- set relativenumber
- set mouse=a
- set inccommand=split
- set tabstop=2
- set shiftwidth=2
- set expandtab
- set confirm
- set autoread

- let mapleader="\<space>"
- nnoremap <leader>; A;<esc>
- nnoremap <leader>C :noh<cr>
